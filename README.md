Next Match
===

App indicator for Linux that shows Red Star fixtures 
---

### Features
* Next football match
* Next basketball match

### Used technologies
* Python 3.5

### Setup
`git clone https://github.com/zsevic/next-match`  
`cd next-match`  
`python3.5 app.py`
